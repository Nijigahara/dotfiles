# ✨Setup✨

[![](assets/demo/showcase.gif)](assets/demo/showcase.gif)

Personal Linux setup managed with `chezmoi`.

## Quick Start

```bash
chezmoi init https://github.com/Nijigahara/dotfiles.git
chezmoi apply
```

Encrypted files require the matching GPG key. See [`docs/secrets.md`](docs/secrets.md).
